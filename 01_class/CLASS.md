Class 객체지향 프로그래밍의 대명사

객체 구성요소
OOP 에서 Object는
- 자바스크립트의 Object 가 아니다.
- 개념적 사상적 접근
- 형체 실체가 없다.

- 행위와 속성으로 객체의 특성을 표현
- 행위 Behavior
- 속성 Attribute : 밥을 먹다, 물을 마시다와 같은 행위의 대상



객체의 구체화
객체를 코드로 구체화하면
- 객체는 클래스가 된다.
- 행위는 메소드가 된다
- 속성은 프로퍼티가 된다.

클래스에
- 메소드와 프로퍼티를 작성
- 클래스 자체로는 사용할 수 없다
- 인스턴스로 생성해야 사용할 수 있다.

= 자바스크립트로 OOP 구현
- 클래스 구조와 구현방법이 다르다.
- Prototype 에 메소드를 연결하는 구조
